Curation Manager

The CurationManager is the mediator between the KnowledgeGraph and the ComputationalModel. It also provides the human-computer interface (HCI) through the Dialog language and IDE.

The Dialog grammar is an extension of the SADL grammar and the Dialog IDE is an extension of the SADL IDE. The objective of Dialog is to provide a controlled-English HCI for directing modeling computations, asking questions, providing information, and browsing the scientific knowledge of the KnowledgeGraph and Computational Model. This is accomplished by providing new grammar constructs that contain the graph-based expressions of the SADL grammar.

Asking Questions

Forms 1-2 are very similar to SADL queries.
Form 3 is a general way to get information about a concept, class, property, or individual, and will return SADL statements that provide information on that concept.
Form 4 is a very general question about property restrictions.
Forms 5-7 are more specific questions about property restrictions.

  1. Ask "<sparql>".                                                                                      Allows any SPARQL query to be executed, SADL syntax without colon after Ask (but OK with)
  2. Select|select ....                                                                                       Queries in SADL syntax but without the leading "Ask:".
  3. What is <concept>?                                                                                Returns what is known about <concept>, reveals enumerated classes, equivalent classes.
  4. What value|values can <prop> of a <class> have?                                    A general question, answer depends on property range and restrictions
    • If there is the cardinality restriction "wheel of bicycle has exactly 2 values."
      • What values of wheel can a bicycle have?                            Answer: "wheel of bicycle has exactly 2 values."
    • If there is the qualified cardinality restriction "digit of hand has exactly 1 value of type thumb."
      • What value can digit of hand have?                                      Answer: "digit of hand has exactly 1 value of type thumb."
    • If there is the all values from restriction "president of a country only has values of type citizen."
      • What value can president of a country have?                        Answer: "president of a country only has values of type citizen."
    • If there is the some values from restriction "president of country has at least one value of type citizen."
      • What value can president of a country have?                        Answer: "president of a country has at least one value of type citizen."
    • If there is the has value restriction "color of snow always has value white."
      • What value can color of snow have?                                   Answer: "color of snow always has value white."
  5. How many values of <prop> can a <class> have?
    • If there is the cardinality restriction "wheel of bicycle has exactly 2 values."
      • How many values of wheel can a bicycle have?                   Answer: "wheel of bicycle has exactly 2 values."
  6. How many values of <prop> of type <class> can a <class> have?  
    • If there is the qualified cardinality restriction "digit of hand has exactly 1 value of type thumb."
      • How may values of digit of type thumb can a hand have?    Answer: "digit of hand has exactly 1 value of type thumb."
  7. What value(s) must <prop> of a <class> have?
    • If there is the some values from restriction "president of a country has at least one value of type citizen."
      • What value must president of a country have?                    Answer: "president of a country has at least one value of type citizen."
    • If there is the has value restriction "color of snow always has value white."
      • What value must color of snow have?                                Answer: "color of snow always has value white."
